Question-1: What are the different ways to select an element in the DOM?

By ID: document.getElementById("myId") By Class Name: document.getElementsByClassName("myClass") By Tag Name: document.getElementsByTagName("p") By CSS Selector: document.querySelector("#myId"), document.querySelectorAll(".myClass")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML: Retrieves or sets the entire HTML content of an element.
InnerText: Retrieves or sets the text content of an element, excluding HTML TextContent: Similar to innerText, but retrieves all text content, including text within script and style tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element to handle events for all its current and future children. This is efficient for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the order in which events are handled in the DOM. When an event occurs on an element, it first triggers event handlers on that element, then on its parent, then its grandparent, and so on, up to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: Use document.createElement("elementName")
Add: Use methods like appendChild(), insertBefore(), or replaceChild() on the parent element.
Remove: Use removeChild() on the parent element.
